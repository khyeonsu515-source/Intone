AI API 설정

Intone 설정

API Key는 이 브라우저의 확장 프로그램 저장소에만 저장됩니다. 웹페이지에서 실행되는 content script에는 전달되지 않습니다.

API Key를 여러 줄로 입력하면 위에서부터 순서대로 사용합니다. 한도가 끝난 키가 있으면 다음 키로 자동 전환합니다.

Cerebras 키도 여러 줄로 입력할 수 있습니다. Groq 키를 모두 사용할 수 없으면 Cerebras 키까지 같은 순환 목록에서 시도합니다.

둘 다 입력하면 분석 전에 같은 URL이 Firebase에 이미 분석되어 있는지 먼저 확인하고, 새로 분석한 결과도 Firebase에 함께 저장해 다른 곳에서도 재사용합니다. 비워두면 이 브라우저 안의 캐시만 사용합니다.

팝업 크기

100%는 현재 사용 중인 90% 크기를 기준으로 합니다.

선택하면 웹페이지 위에 뜨는 분석 팝업 크기에 바로 적용됩니다.
